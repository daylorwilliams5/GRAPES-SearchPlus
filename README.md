# GRAPES Search+

## Overview
GRAPES Search+ is a natural-language search assistant for UCLA’s Division of Graduate Education fellowship database.  
UCLA already has a GRAPES website where students can search for funding, but it is slow, unintuitive, and often fails to return useful results. GRAPES Search+ solves this by letting students type queries in plain English and instantly receive relevant fellowship opportunities.  

## Motivation
I built this project to address a common frustration at UCLA: the official GRAPES website exists, but students rarely use it because it hardly works. By making fellowship search faster and more reliable, GRAPES Search+ helps students actually take advantage of the opportunities available to them.  

## Features
- 🔎 **Natural Language Search** – search fellowships using everyday language  
- 📊 **JSON/Excel Data Parsing** – integrates directly with UCLA’s structured database  
- 🌐 **DuckDuckGo Integration** – supplements results with external fellowship information  
- 🖥️ **User-Friendly Interface** – fast and lightweight HTML interface

## Tech Stack
- Python (pandas, regex, argparse) for data enrichment  
- DuckDuckGo Search API for URL expansion  
- JSON for structured data storage  
- HTML/CSS/JavaScript for the frontend  
- GitHub Pages for deployment
